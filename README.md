# 🔐 NetRecon Pro – Cyber Security Toolkit

A Python-based cybersecurity reconnaissance toolkit with GUI that performs network scanning, port scanning, and generates security reports.

---

## 🚀 Features

- 🌐 Network device discovery using ARP scanning
- 🖥️ Displays live devices (IP + MAC addresses)
- 🔎 TCP Port scanning (common ports)
- 💾 Scan history stored using SQLite database
- 📄 Generates professional PDF security reports
- 🖱️ Simple GUI interface using Tkinter

---

## 🧰 Technologies Used

- Python
- Scapy
- Tkinter (GUI)
- SQLite (Database)
- ReportLab (PDF generation)
- Socket programming
